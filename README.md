# Prompt-Powered Go

A beginner-friendly Go (Golang) toolkit built with AI assistance.  
This project demonstrates a minimal HTTP server with two endpoints.

## Features
- `/` → Returns "Hello, Go!"
- `/health` → Simple health check

## Requirements
- Go installed

## Setup & Run
```bash
go run main.go
```

Open in your browser:

http://localhost:8080/

http://localhost:8080/health
